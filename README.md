## 📌 Sobre

Este repositório é o meu "caderno de anotações" de código: aqui reúno tudo que venho praticando durante o aprendizado de Java, desde a lógica de programação até conceitos mais avançados como Programação Orientada a Objetos.

Não é um projeto de produção — é um espaço de **aprendizado contínuo**, então o código evolui (e às vezes se refaz) conforme os conceitos vão sendo consolidados.

## 🗂️ Módulos

O conteúdo está organizado por módulo, seguindo a progressão do curso:

- **Módulo 1** — Lógica de Programação
- **Módulo 2** — Programação Orientada a Objetos
- **Módulo 3** — *(em andamento)*

## 🛠️ Tecnologias

- **Linguagem:** Java
- **IDE:** IntelliJ IDEA

## 🚀 Como executar

Cada exercício pode ser compilado e executado individualmente com o Java instalado na máquina:

```bash
javac NomeDoArquivo.java
java NomeDoArquivo
```

Ou, se preferir, basta abrir o projeto diretamente na IDE de sua preferência.

## 🎯 Objetivo

Manter um histórico de evolução no aprendizado de Java, servindo tanto como material de revisão pessoal quanto como portfólio do progresso ao longo do curso.
